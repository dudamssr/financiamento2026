#  Simulador de Financiamento

##  Descrição do Projeto

Devido às altas taxas de juros no cenário atual, a aquisição de bens e serviços por meio de financiamento se torna mais difícil. Antes de realizar a compra de itens como carros, motos, imóveis ou eletrodomésticos, é essencial compreender o impacto dos juros no valor final pago.

Este projeto consiste em um aplicativo que permite simular financiamentos de forma simples e prática.

O usuário informa:
- Valor do bem  
- Número de parcelas  
- Taxa de juros mensal  
- Taxas adicionais  

E o sistema retorna:
- Valor da parcela  
- Montante total do financiamento

##  Como Executar o Projeto

1. Clone o repositório:
```bash
git clone https://github.com/seu-usuario/simulador-financiamento.git
```

2. Acesse a pasta do projeto:
```bash
cd simulador-financiamento
```

3. Instale as dependências:
```bash
flutter pub get
```

4. Execute o aplicativo:
```bash
flutter run
```

---

##  Funcionamento do Aplicativo

1. Insira o valor do bem desejado  
2. Informe o número de parcelas  
3. Adicione a taxa de juros mensal  
4. Inclua taxas adicionais (se houver)  
5. O sistema calculará automaticamente:
   - Valor de cada parcela  
   - Valor total a ser pago
